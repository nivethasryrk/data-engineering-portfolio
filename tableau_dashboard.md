# Tableau Dashboard Example

## Objective
To create interactive dashboards for business insights and reporting.

## Example Use Case
Sales Dashboard for business performance tracking.

## Key Features

- Revenue trends over time
- Sales by region
- Top customers and products
- Filters for dynamic analysis
- Drill-down capabilities

## Tools Used

- Tableau
- SQL for data extraction
- Data warehouse (Redshift / Synapse)

## Dashboard Components

- Line charts for trends
- Bar charts for comparisons
- KPIs for summary metrics
- Filters for user-driven insights

## Business Impact

- Enabled business users to track performance in real time
- Improved decision-making through visual insights
- Supported self-service analytics

## Interview Summary

“I built Tableau dashboards with filters and KPIs to help business users analyze sales trends, customer behavior, and performance metrics.”
